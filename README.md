# Cycle Website

A clean and energetic landing page designed for cycling enthusiasts. Focused on bold visuals, smooth layout, and an active outdoor vibe.

## Features

- **Bold Visuals**: Eye-catching design that captures the energy of cycling
- **Smooth Layout**: Intuitive navigation and seamless user experience
- **Active Outdoor Vibe**: Modern aesthetic celebrating cycling culture
- **Responsive Design**: Works beautifully on all devices

## Tech Stack

- React with TypeScript
- Vite for fast development
- Tailwind CSS for styling
- Modern component architecture

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   `bash
   git clone https://github.com/Sahilarora83/cycle-website.git
   cd cycle-website
   `

2. Install dependencies:
   `bash
   npm install
   `

3. Start the development server:
   `bash
   npm run dev
   `

4. Open your browser and navigate to the local server URL (typically http://localhost:5173)

## Project Structure

`
 components/        # React components
 src/
    components/   # Additional component styles
    images/       # Image assets
 App.tsx           # Main app component
 index.tsx         # Entry point
 constants.ts      # App constants
 types.ts          # TypeScript type definitions
`
